Cómo instalar los códigos QR

Al crear tu cuenta has agregado varios Checkpoints, recibirás un email con tódos los códigos QR, pero además puedes descargarles desde el propio panel de gestión, puedes entrar a la plataforma de gestión gratuita.

  1. Imprima los códigos QR generados.
  2. Recorte cada código QR individualmente.
  3. Le recomendamos que plastifique los códigos QR,o les envuelva en cinta transparente para protegerlos.
  4. Limpie la superficie donde va a realizar la instalación.
  5. Pegue el código QR con cinta adhesiva de doble cara.
  6. Asegúrese de que el código QR sea visible y accesible.

Coloque los códigos QR entre 1.2 y 1.5 metros del suelo para un escaneo cómodo.

Descarga nuestra aplicación para realizar las rondas

Los diferentes controladores o patrulleros, podrán utilizar cualquier terminal móvil para realizar la lectura de los códigos QR.
Es tán sencillo como abrir una simple página web, o descargar nuestra aplicación móvil.

web Página web Solo hay que entrar en el enlace Página web control de rondas y escanear los códigos QR. smartphone Aplicación móvil Descarga nuestra aplicación móvil desde Google Play o App Store para un acceso más rápido y cómodo. monitor-smartphone Aplicación PWA Dentro de la Página web control de rondas, puedes instalar la aplicación como una PWA (Progressive Web App) para un acceso más rápido y cómodo.

check La aplicación es compatible con cualquier dispositivo móvil con cámara y conexión a internet.

Quieres que enviemos instrucciones a los controladores? Simplemente pulsa en el botón y les enviaremos un email con las instrucciones de uso. Es necesario que hayas registrado una dirección de email a cada uno de ellos.

Tres sencillos pasos para marcar puntos de control

  1. Accedemos a la web o a la aplicación
  2. El controlador debe registrar su identificador o email. No necesitan contraseña para reportar.
  3. Escanea los código QR según va avanzando en la ronda
  4. Opcionalmente: Se puede configurar acciones o listas de control en cada uno de los puntos.
